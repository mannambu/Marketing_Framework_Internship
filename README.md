皆さん、お疲れ様です。
今日は1ヶ月間のインターンシップで経験したいと思います。

まず、生活についてです。
生活にはだいぶ慣れてきて、とても快適に過ごしています。去年際に、
2週間日本で見学することがあっておかげで、生活および雰囲気にある程度慣れていました。
そのため、食事やゴミ分別なども特に大きな問題はありません。

次、週末の経験についてです。
毎週、色々な場所へ出かけました。新潟市内だけではなく、
新潟県内の色々な場所を訪れることができて、とてもいい経験になりました。

その中でも、一番印象に残っている場所は「清津峡Tunnel of Light」です。
インターネットで調べると、「新潟県に来たらぜひ一度は行ったほうがいい場所」と紹介されていました。
実際に行ってみると、とても景色がきれいで感動しました。

しかし、一つ不便だと感じていることがあります。
それは、バスの本数が少なかったので、時間を合わせるのが少し大変でした。

最終に、仕事についてです。
仕事を始めたばかりのころは、日本語で仕事をしながら、
会社のスケジュールに合わせて作業を進めなければならなかったので、
少し大変だと感じていました。

しかし、1か月が経って、その環境にも少しずつ慣れてきました。
もちろん、まだ間違えることもありますが、最初のころと比べると、
自分でも成長できたと感じています。

また、Toànさんと坂井さんには、本当にたくさん助けていただきました。
仕事の進め方だけでなく、いろいろなことを丁寧に教えていただき、
とても感謝しています。

今回のインターンシップを通して、学校では学べないことをたくさん経験することができました。
実際に働くことで、自分自身も少し成長できたと思います。

これからも、もし機会があれば、もっといろいろな仕事に挑戦して、
多くのことを学び、自分の力をさらに伸ばしていきたいと思います。

ご清聴ありがとうございました。
